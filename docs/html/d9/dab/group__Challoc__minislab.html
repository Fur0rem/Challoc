◆ __attribute__()

__attribute__ ( (aligned(4096)) )

Minislab to allocate small memory regions It contains 512, 256, 128, 64, 32, 16, 8 and 4 bytes layers and keeps track of the usage of each layer using bitmasks.

Chunks of memory

< 1 chunk of 512 bytes

< 2 chunks of 256 bytes

< 4 chunks of 128 bytes

< 8 chunks of 64 bytes

< 16 chunks of 32 bytes

< 32 chunks of 16 bytes

< 64 chunks of 8 bytes

< 64 chunks of 4 bytes

Bitmasks to keep track of the usage of each layer

< Usage bitmask of 4 bytes chunks

< Usage bitmask of 8 bytes chunks

< Usage bitmask of 16 bytes chunks

< Usage bitmask of 32 bytes chunks

< Usage bitmask of 64 bytes chunks

< Usage bitmask of 512, 256 and 128 bytes chunks combined for compactness
